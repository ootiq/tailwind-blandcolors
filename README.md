# tailwind-blandcolors
It just adds the black transparency colors.

### Colors
```
bg-bland-100: rgba(0, 0, 0, 0.1)
bg-bland-200: rgba(0, 0, 0, 0.2)
...
```

## Install
```
yarn add -D @ootiq/tailwind-blandcolors
```

## Usage
```html
<div class="bg-bland-100" >

</div>
```
